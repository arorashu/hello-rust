# Learning Rust

Tutorial: (https://stevedonovan.github.io/rust-gentle-intro/readme.html)

## How to Run
You should have `rust` installed.
Use this tutorial to install [Rust](!https://www.rust-lang.org/tools/install)

    # compile the rust file
    $ rustc hello.rs
    # run the executable
    $ ./hello
